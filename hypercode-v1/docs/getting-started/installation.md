# Installation

## Prerequisites

- Python 3.8 or later
- pip (Python package manager)

## Installation Methods

### Using pip

```bash
pip install hypercode
```

### From Source

```bash
# Clone the repository
git clone https://github.com/welshDog/hypercode.git
cd hypercode

# Install in development mode
pip install -e ".[dev]"
```

## Verifying Installation

```bash
hypercode --version
```
