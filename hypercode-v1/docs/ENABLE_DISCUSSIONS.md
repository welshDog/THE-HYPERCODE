# Enable GitHub Discussions - Quick Guide

## Step 1: Enable Discussions in Repository Settings

1. Go to https://github.com/welshDog/hypercode/settings
2. Scroll down to "Features" section
3. Check the "Discussions" checkbox
4. Click "Save changes"
