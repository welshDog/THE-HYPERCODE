# DuelCode Tutorials

Welcome to the DuelCode tutorials! These guides will help you understand and use HyperCode's dual representation system effectively.

## Available Tutorials

### 1. Counter Application
A beginner-friendly introduction to DuelCode principles through building an interactive counter.

- **Location**: `/examples/duelcode/counter_tutorial_v2.md`
- **Prerequisites**: Basic programming knowledge
- **Concepts Covered**:
  - Dual representation (visual + code)
  - Interactive UI components
  - Event handling
  - State management

## What is DuelCode?

DuelCode is HyperCode's approach to making programming more accessible through dual representation:

- **Visual + Code**: See both representations simultaneously
- **Interactive**: Modify and see changes in real-time
- **Accessible**: Designed with neurodivergent thinkers in mind

## Getting Started

To try out the tutorials:

2. Open `counter_tutorial_v2.md` in your preferred markdown viewer
3. Follow along with the interactive examples
