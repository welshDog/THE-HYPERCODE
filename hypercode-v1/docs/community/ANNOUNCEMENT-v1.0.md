## üöÄ Getting Started

### **Quick Install**

```bash
git clone https://github.com/welshDog/hypercode.git
cd hypercode
pip install -e .
```

### **Development Setup**

```bash
# Install development dependencies
pip install -r config/requirements-dev.txt

# Set up pre-commit hooks
pre-commit install

# Run tests
pytest
```

### **VS Code Extension**

```bash
# Install spatial visualizer
cd spatial_visualizer
npm install
npm run compile
```
