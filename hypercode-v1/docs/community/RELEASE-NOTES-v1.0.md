## üîß Installation

### Quick Start

```bash
git clone https://github.com/welshDog/hypercode.git
cd hypercode
pip install -e .
```

### Development Setup

```bash
pip install -r config/requirements-dev.txt
pre-commit install
pytest
```

### VS Code Extension

```bash
cd spatial_visualizer
npm install
npm run compile
```
