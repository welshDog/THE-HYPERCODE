# HyperCode Examples

This directory contains various examples demonstrating HyperCode's features and capabilities.

## Getting Started

### Basic Examples
- `hello_world.hc` - The classic "Hello, World!" program
- `demo_hello.hc` - Simple demonstration of basic I/O
- `fibonacci.hc` - Fibonacci sequence implementation
- `demo_fibonacci.hc` - Interactive Fibonacci number generator

### Language Features
- `oop.hc` - Object-oriented programming concepts
- `first_click_moment.hyper` - Interactive example showing HyperCode's unique features
- `task_manager.hc` - Task management system demonstrating data structures

### Advanced Topics
- `quantum_computing.hc` - Quantum computing concepts
- `dna_computing.hc` - DNA computing simulation
- `spatial_computing.hc` - Spatial computing examples

### Web Development
- `web_server.hc` - Basic web server implementation
- `todo_web_app.hc` - Full-stack todo application
- `web_development.hc` - Web development patterns

### Data Processing
- `data_analysis.hc` - Data analysis examples
- `data_visualization.hc` - Data visualization techniques
- `database_integration.hc` - Database connectivity examples

### Creative Demos
- `creative-demos/fractal-explorer/` - Interactive fractal visualization
  - `fractal.hc` - Fractal generation code
  - `index.html` - Web interface
  - `README.md` - Setup instructions

## Running Examples

To run any example:
```bash
python -m hypercode examples/path/to/example.hc
```

For web examples, additional setup may be required. Refer to individual example documentation for details.

## Contributing

We welcome contributions! To add a new example:
1. Create a new `.hc` file in the appropriate directory
2. Add clear comments and documentation
3. Update this README with a brief description
4. Submit a pull request
